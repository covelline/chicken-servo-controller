# 概要

[NT東京2024](https://nt-tokyo.org/)に出展した[びっくりチキンをMIDIキーボードに繋いでみた](https://nt-tokyo.org/exhibitors/113)のMIDI入力を受け取りサーボを動かすコード

必要なもの

- ラズパイ
- usb接続できるmidiキーボード
- サーボ(TD-8125MG)
- サーボドライバ(PCA9685)
